# 🍽️ La Famiglia Restaurant Website

A modern and responsive family restaurant website built using **HTML, CSS, and JavaScript**, featuring online ordering and table reservation functionality.

---

## 🌟 About La Famiglia

**"Where Every Meal Tells a Story"**

La Famiglia is a family-owned Italian restaurant inspired by generations of authentic recipes. What started as *Nonna Rosa's kitchen* has grown into a warm and welcoming dining experience where food brings people together.

---

## ✨ Features

### 🎨 Modern UI/UX
- Clean, elegant, and responsive design  
- Family-friendly layout with smooth navigation  

---

### 🍽️ Online Ordering System
- Browse menu items  
- Add/remove items dynamically  
- Real-time total price calculation  
- Order form with user details  
- Confirmation message using JavaScript  

---

### 📅 Table Reservation System
Users can:
- Enter name, email, and phone number  
- Select date and time  
- Choose number of guests  
- Select occasion (birthday, anniversary, etc.)  
- Add special requests (dietary needs, decorations)  

✅ Includes **JavaScript form validation** and booking confirmation  

---

### 🖼️ Gallery Section
- Showcases food, ambience, and restaurant experience  

---

### ⭐ Reviews Section
- Displays customer testimonials and ratings  

---

### ⚡ JavaScript Functionalities
- Form validation  
- DOM manipulation  
- Event handling  
- Dynamic cart system  
- Interactive UI updates
